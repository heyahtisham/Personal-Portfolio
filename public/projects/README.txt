Project thumbnails
==================

Save your screenshots into this folder with these exact names:

  personal-portfolio.png   → used by "Personal Portfolio" and "React Web Applications"
  landing-page.png         → used by "Landing Page Collection"

PNG or JPG both work (if you use .jpg, update the paths in
src/data/projects.ts). Recommended: ~1600px wide, 16:10-ish crop of
the top of the page. Until a file exists, the card automatically
shows the abstract mockup instead.
